# 高市総理 支持率コレクター

## ゲーム概要

高市総理大臣を操作する3D TPS（サードパーソンシューティング）形式のモバイルゲームです。

## プラットフォーム

- iOS
- Android

## ゲームルール

1. プレイヤーは高市総理大臣キャラクターを操作します
2. フィールド上に散らばる「支持率」アイコンを収集します
3. 制限時間内にできるだけ多くの支持率を獲得することが目標です
4. 獲得した支持率の数がスコアとなります

## 基本操作

- **移動**: 仮想ジョイスティック（画面左側）
- **カメラ操作**: スワイプ（画面右側）

## ゲームの流れ

1. タイトル画面 → ゲーム開始
2. 制限時間内にフィールドを走り回り支持率を収集
3. 時間終了後、獲得した支持率数を表示
4. リトライまたはタイトルに戻る
